Objetivo

O Laboratório 02, realizado pelos integrantes César Augusto Brilhante Teixeira, Gabriel Silva Cajueiro de Campos, Mayara Maria Triveloni tem por objetivo a aplicação prática dos conceitos de Smoothing Images trazido em discplina.

Para a realização deste laboratório, foi utilizado o Notepad++ configurado em linguagem HTML para a criação do código da disciplina as imagens geradas pela equipe no laboratório 1 relacionado a cada membro. O procedimento do laboratório será a aplicação dos códigos de Smoothing Images deste link.

Em seguida, para cada um dos capítulos posteriores, haverá uma explicação do método utiilizado, bem como o entendimento prático das alterações no mesmo. Será feita uma explicação com o exemplo da foto de um integrante do grupo e, posteriormente, as imagens seguintes irão ser a replição desde recorte de código, exceto com a alteração do caminho de foto.


Aplicação de 2D Convolution ( Image Filtering )

De modo geral, durante a prática foram utilizados filtros para a realização do desfoque da imagem e filtros personalizados, como é o caso do '2D Convolution'.

A seguir, é possível analisar o código utilizado para gerar as imagens, aplicando a função cv.filter2D(), fornecida pela biblioteca Open CV. Esta função aplica um filtro linear arbitrário na imagem.

No site disponibilizado no 'Objetivo', há uma explicação para os valores utilizados no código, como por exemplo, a escolha e demonstração do kernel de filtro 5x5.

Para obter o efeito desejado da técnica Gaussian Blurring, a função aplicada no código é o cv.GaussianBlur(), sendo este um desfoque eficaz na remoção do ruído gaussiano de uma imagem, onde especifica-se a largura e altura do kernel (no caso, positivo e ímpar).
