Lab 01

Objetivo

O Laboratório 01, realizado pelos integrantes César Augusto Brilhante Teixeira, Gabriel Silva Cajueiro de Campos, Mayara Maria Triveloni tem por objetivo a manipulação de códigos em Python através de Comandos em terminal para alteração de parâmetros de imagem e vídeo, tanto para imagens geradas em câmera, como para imagens fornecidas.

Para a realização deste laboratório, foi utilizado o Notepad++ configurado em linguagem HTML para a criação do código da disciplina e um editor simples de imagem para as manipulações mais básicas relacionadas a montagens de fotos dos integrantes da equipe.

Manipulação de Imagem

Na linha 4 em "img = cv.imread('messi5.jpg',0)", o parâmetro numérico 0 define se a imagem será ou não em preto e branco.

Manipulação de Frame de Vídeo

Na linha 15 em "time.sleep(1/25.0)", o numero de frames por segundo é definido pelo valor dividido por 1, para acelerar o vídeo, seria apenas aumentar o valor de 25 para 40 e o contrário para diminuir a velocidade.

Salvamento de Imagem

Selecionada está a rotina de salvamento. e a de fechamento do programa. Ambas as rotinas interrompem o loop while e levam a aplicação a fechar. No caso a rotina de fechamento com o botão q foi replicada, substituindo o botão q pelo x e inserindo a função cv.imwrite para salvar os frames da foto no PC.

Inversão de Vídeo

Está selecionada a função de reversão do vídeo que originalmente está comentada no código. A mudança para que o vídeo seja salvo corretamente é retirar essa linha de comentário.
